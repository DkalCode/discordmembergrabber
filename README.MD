## About

A program to export all member names and ids to a file.

## Installation

Clone from Git or download manually  
```bash
$ git clone https://github.com/DkalCode/discordmembergrabber.git
```

Install the Dependencies 
```bash
$ npm install
```

Run the Script 
```bash
$ npm start
```

## Configuration
